Standard Mirage Input Component - Pure CSS Implementation

HTML Usage:
<div class="mir-input-group">
  <input type="tel" id="mobile" name="mobile" class="mir-input" placeholder=" " autocomplete="tel">
  <label class="mir-label" for="mobile">شماره موبایل</label>
  <i class="mir-input-icon" data-feather="phone"></i>
</div>

Key Features:
• placeholder=" " (single space) is required for CSS :not(:placeholder-shown)
• Pure CSS implementation - no JavaScript required
• Works with autofill, pre-filled values, and all modern browsers
• RTL optimized for Persian/Arabic languages
• This is the standard input style for Mirage Design System

راهنمای استفاده - Usage Guide

<!-- Import CSS --> <link rel="stylesheet" href="components/input/input.css"> <!-- Import JavaScript (Optional - only for validation/Persian numbers) --> <script src="components/input/input.js"></script> <!-- Standard Mirage Input --> <div class="mir-input-group"> <input type="text" id="name" class="mir-input" placeholder=" "> <label class="mir-label" for="name">نام</label> </div> <!-- Input with Validation (JavaScript) --> <script> // Setup validation const nameInput = document.querySelector('#name'); new MirageInput(nameInput, { validation: { required: true, minLength: 2 } }); // Validate form const form = document.querySelector('#myForm'); const isValid = MirageInput.validateForm(form); </script>
